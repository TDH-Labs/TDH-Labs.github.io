# TDH Labs Product Template

This is a template for new TDH Labs products. Copy this directory when starting a new project.

## Structure

```
product-name/
├── src/              # Source code
├── tests/            # Tests
├── docs/             # Documentation
├── scripts/          # Build/deploy scripts
├── .github/          # CI workflows
│   └── workflows/
├── README.md         # Product documentation
├── LICENSE           # License file
└── pyproject.toml    # Python project config (or equivalent)
```
